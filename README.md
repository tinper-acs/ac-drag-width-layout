# 可拖拽列宽

## 如何使用
使用 npm 安装最新版本

```
$ npm install ac-drag-width-layout  --save

引入
import AcDragWidthLayout from 'ac-drag-width-layout';
样式
import 'ac-drag-width-layout/dist/index.css';

```
